Selector de tipo

Se refiere al tipo de etiqueta por ejemplo div, p y ul son elementos de diferente tipo.

<p>Hola me llamo Wendy</p>

Formato CSS:

p { color: mediumslateblue; }

El resultado es este:

Hola me llamo Wendy

Seleccionamos los elementos con un id especifico.

#aviso { color: rgb(201, 193, 255); font-size: 200%; }

El resultado es este:

Texto2

Selector de hijos

Este selecciona elementos que son hijos directos de otros elementos.
Un elemento hijo es cualquier elemento que es anidado directamente de otro elemento.

<p><span>Texto</span></p>
<p><a href="#">Texto</a></p>
<p><span>Texto3</span></p>


Formato CSS:

p>span {color: firebrick;}

El resultado es este:

Texto

Texto2

Texto3
