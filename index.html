Selector de tipo

Se refiere al tipo de etiqueta por ejemplo div, p y ul son elementos de diferente tipo.

<p>Hola me llamo Wendy</p>

Formato CSS:

p { color: mediumslateblue; }

El resultado es este:

Hola me llamo Wendy

Selector por ID

Seleccionamos los elementos con un id especifico.

<p id="aviso"> Texto1 </p>
<p>Texto2 </p>

Formato CSS:

#aviso { color: rgb(201, 193, 255); font-size: 200%; }

El resultado es este:

Texto1

Texto2

Selector descendente

Selecciona un elemento dentro de otro elemento

<div id="descendente">
<span>Texto1</span>
</div>

Formato CSS:

#descendente span { font-size: 20px; color: crimson; }

El resultado es este:

Texto1

Selector de clase

Selecciona elementos por su clase

<p class="destacado">texto</p>
<p>texto2</p>

Formato CSS:

.destacado { font-family: cursive; }

El resultado es este:

Texto1

Texto2

Selector universal

Puedes seleccionar todos los elementos con el selector universal

<ul>
<li>Hola<li>
<li>Wapos<li>
<ul>

Formato CSS:

ul * {font-size: large; color: yellowgreen;}

El resultado es este:
  • Hola
  • Wapos

Selector adyacente

Este selecciona los elementos que le siguen de un primer elemento

<h1>Titulo</h1>
<h2>Subtitulo</h2>
<h2>Otro subtitulo</h2>

Formato CSS:

h1+h2 {color: lightcoral;}

El resultado es este:

Titulo

Subtitulo

Otro subtitulo

Selector general de hermanos

Este selecciona todos los hermanos de un elemento que le sigue a este

<h5>Buenos</h5>
<h6>días</h6>

Formato CSS:

h5~h6 {font-size: 35px; font-family: fantasy;}

El resultado es este:
Buenos
días

Selector de hijos

Este selecciona elementos que son hijos directos de otros elementos.
Un elemento hijo es cualquier elemento que es anidado directamente de otro elemento.

<p><span>Texto</span></p>
<p><a href="#">Texto</a></p>
<p><span>Texto3</span></p>


Formato CSS:

p>span {color: firebrick;}

El resultado es este:

Texto

Texto2

Texto3

Pseudo-selector primer hijo

Selecciona un primer elemento hijo que está dentro de otro elemento.

<div class="prueba">
<p>texto1</p>
<p>texto2</p>
<p>texto3</p>
</div>

Formato CSS:

.prueba p:first-child { background-color: cornflowerblue; border-radius: 3px;}

El resultado es este:

texto1

texto2

texto3

Only-child pseudo-selector

Selecciona un elemento que sea el único elemento dentro de otro.

<div class="prueba2">
<p id="texto1">texto1</p>
<p>texto2</p>
</div> <div class="prueba2">
<p id="texto3">texto3</p>
</div>

Formato CSS:

.prueba2 #texto1, #texto3:only-child { background-color: darkblue; color: coral; }

Este es el resultado:

texto1

texto2

texto3

Last-child pseudo selector

Selecciona el último elemento dentro de otro elemento.

<div class="prueba3">
<div><p class="texto">Texto1</p></div>
<p>un espacio</p>
<p>otro espacio</p>
<p class="texto">Texto2</p>
</div>

Formato CSS:

.texto:last-child {font-family: cursive; color: hotpink;}

El resultado es este:

Texto1

un espacio

otro espacio

Texto2

First of type selector

<div class="prueba6">

<li>lista3</li>
